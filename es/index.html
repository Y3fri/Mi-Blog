PROYECTOS

Desarrollé este proyecto usando una API en C# con Entity Framework para conectarla a una base de datos en SQL Server. Decidí implementar Swagger para tener una documentación interactiva y utilicé SignalR para actualizar los datos en tiempo real, lo cual resultó muy útil.

Para asegurar una autenticación robusta, integré JWT, lo que nos permitió manejar la seguridad de manera efectiva.

En cuanto al diseño del front-end, elegí Angular para construir las vistas de la aplicación. Me beneficié de Angular Material para aprovechar sus componentes predefinidos, lo que facilitó mucho la creación de interfaces visualmente atractivas y funcionales.

En la página web que diseñé, se muestran productos reales en lugar de datos aleatorios. Puedes agregar productos al carrito, y algunos de ellos tienen descuentos especiales. También incluí una lista de categorías para que puedas filtrar los productos según tus intereses. Además, añadí un botón para llamar al mesero en tiempo real, lo cual facilita mucho la comunicación si necesitas algo mientras navegas por la página.

La página tiene secciones de administración donde puedo crear, ver, actualizar y eliminar sedes, productos, categorías y usuarios con roles específicos como mesero, cocinero y administrador de sede.

PWA WEB

He trabajado en un proyecto que consiste en una PWA (Aplicación Web Progresiva) diseñada para que los usuarios puedan descargar la aplicación en sus teléfonos. El propósito de esta aplicación es facilitar la recolección de materiales reciclables directamente desde las casas de los usuarios, quienes pueden intercambiar estos materiales por puntos que luego pueden canjear por productos de limpieza.

El objetivo principal de este proyecto es concienciar a la gente sobre la importancia del reciclaje sin necesidad de salir de sus hogares. Aunque la aplicación ya está desarrollada, aún no está en producción.

Este es un formulario para solicitar la recogida de materiales reciclables. Los usuarios deben ingresar sus datos personales y su ubicación. También se utilizó una API de Google Maps para especificar el lugar exacto de la recogida, permitiendo que el servicio localice con precisión la dirección de recolección.

La aplicación también incluye una sección administrativa que ofrece diversas tablas para consultar, crear, editar y eliminar datos. Aquí se pueden gestionar productos para el canje de puntos, registrar usuarios (tanto trabajadores como administradores), así como programar fechas y horarios para la recogida de materiales y otras funciones relacionadas.

Aquí están las imágenes de entrenamiento, que representan las vocales en mayúsculas. Cada imagen es de 7x5 píxeles, donde el negro es 1 y el blanco es 0.

La red neuronal artificial (RNA) que estoy usando tiene 35 entradas (una por cada píxel), una capa oculta con 3 neuronas y una capa de salida con 3 neuronas. Las salidas se codifican de la siguiente manera:

  • A: 000
  • E: 100
  • I: 110
  • O: 001

RNA BACKPROPAGATION

Esta es una red neuronal artificial (RNA) para identificar peces. A la izquierda tenemos el pez trucha y a la derecha un pez más colorido, el Pomacanthus navarchus. La idea es identificar estos peces.

Primero, realicé un tratamiento de imagen a las fotos de los peces, pasándolas por canales azules, rojos o verdes. Luego, traté de quitarles el fondo y aplicarles un kernel para que las imágenes quedaran como se muestra a continuación. Lo que buscamos es extraer el contorno del pez y segmentarlo para identificarlo.

En la parte de entrenamiento, procesamos la imagen reduciéndola a 70x70 píxeles, es decir, 4900 entradas. La RNA tiene una capa oculta con 7 neuronas y una capa de salida con 1 neurona. Las salidas están codificadas de la siguiente manera: si es 0, es el pez Pomacanthus navarchus, y si es 1, es el pez trucha.

Aquí está la demostración con otras imágenes de los peces. Podemos ver que la red neuronal los identifica muy bien y se aproxima a las salidas deseadas. En este caso, se busca el 0, pero nos da 0.3618.

En este otro caso, como es trucha, debería aproximarse a 1, pero nos da una salida de 0.8631.
